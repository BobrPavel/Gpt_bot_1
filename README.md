# Бот GPT
Это Telegram бот с интеграцией chatgpt. Выполнялся в качестве тестового задания и для того, чтобы опробовать 
библиотеку openai.


## Стек технологий
- Python 3.11
- aiogram 3
- openai

## Интеграции
- Chat GPT 4


## Локальное развёртывание проекта
Клонируйте репозиторий проекта и перейдите в него с помощью командной строки
```
git clon https://github.com/BobrPavel/Gpt_bot_1.git
```

Создайте виртуальное окружение Pyton и активируйте его
```
python3 -m venv venv  
```
```
venv\Scripts\activate  
```
Установите зависимости

```
pip install -r requirements.txt  
```
Создайте файл .env и запишите туда ваш Telegram токен и AI API key/
```
TOKEN=Ваш токен
OPENAI_KEY=Ваш GPT API KEY
```

## Запуск бота:
Запустите файл bot.py и любое приватное сообщение боту.
